# v0.5.0 - Financial Depth (Future)

> These are planned future enhancements. None are implemented yet unless noted.

## US#12: Tax Calculations

Calculate income tax on pension withdrawals.

- Personal allowance: £12,570
- Basic rate: 20% (£12,571–£50,270)
- Higher rate: 40% (£50,271–£125,140)
- Show net (after-tax) income alongside gross
- Tax-free lump sum already implemented (see US#2)

## US#13: State Pension Enhancements

Extend state pension modelling beyond current implementation.

- **Already implemented** (US#8): Qualifying years slider, proportional calculation, start age
- **Planned**: Triple Lock modelling (2.5% minimum annual increase)
- **Planned**: Deferral bonus (5.8% per year deferred)
- **Planned**: State pension forecast import

## US#14: Multiple Pots

Manage ISA, SIPP, Workplace pensions separately.

- Support 10+ pots
- Individual growth rates per pot
- Tax treatment per pot type (ISA tax-free, SIPP marginal rate)
- Aggregate view across all pots

Compare retirement income strategies.

- Level / escalating / inflation-linked annuity quotes
