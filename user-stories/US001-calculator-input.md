# US#1: Basic Calculator Input Form

**As a** UK pension saver  
**I want to** input my current pension details  
**So that** I can see projections for my retirement

## Acceptance Criteria
- [x] Input field for date of birth (DD/MM/YYYY)
- [x] Input field for planned retirement date
- [x] Input field for current pension pot (£)
- [x] Input field for monthly contribution (£)
- [x] All fields have validation
- [x] Clear error messages for invalid inputs

## Technical Implementation
- HTML5 date inputs
- Number inputs with currency formatting
- Min/max validation (age 18-75, pot £0-£10M)

- `src/index.html`

- `src/style.css`
